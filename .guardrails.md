# 🛡️ Guardrails - ป้องกัน AI มักง่าย

## วัตถุประสงค์

ป้องกันไม่ให้ AI แก้ไฟล์ Test/Config เพื่อให้ผ่านการทดสอบแบบมักง่าย บังคับให้แก้ Code Logic จริงๆ

---

## ไฟล์/โฟลเดอร์ที่ถูกป้องกัน

### 📋 รายการ:

| ประเภท | รายการ | เหตุผล |
|--------|---------|---------|
| **Test Files** | `*.test.js`, `*.test.ts`, `*.spec.js`, `*.spec.ts` | ป้องกันแก้ Test ให้ผ่าน |
| **Test Folders** | `__tests__/**`, `tests/**`, `spec/**` | ป้องกันแก้ Test แบบมักง่าย |
| **Test Config** | `jest.config.js`, `vitest.config.js`, `playwright.config.js`, `cypress.config.js` | ป้องกับแก้ config เพื่อให้ผ่าน |

---

## ⚠️ การทำงาน

### เมื่อคุณ (หรือ AI) พยายาม commit ไฟล์ที่ถูกป้องกัน:

```bash
git add broken.test.js
git commit -m "fix: make tests pass"
```


---

## ✅ วิธีใช้งานอย่างถูกต้อง

### **ถ้า Test ล้มเหลว:**

1. **อย่า** แก้ไฟล์ Test ให้ผ่าน
2. **แก้** Code Logic จริงๆ
3. **รัน Test** อีกครั้ง
4. **ถ้า Test ผิดจริง** ค่อยแก้ Test พร้อมคำอธิบาย

## 🔧 การติดตั้ง

### 1. ก๊อปปี้ Hook Script:

```bash
# Linux/Mac
cp .git/hooks/pre-commit-hook.sh .git/hooks/pre-commit
chmod +x .git/hooks/pre-commit

# Windows (Git Bash)
cp .git/hooks/pre-commit-hook.sh .git/hooks/pre-commit
chmod +x .git/hooks/pre-commit
```

### 2. หรือใช้ Husky (ถ้ามี):

```json
// package.json
{
  "husky": {
    "hooks": {
      "pre-commit": "bash .git/hooks/pre-commit-hook.sh"
    }
  }
}
```

## 🚨 Exit Codes

| Exit Code | ความหมาย |
|-----------|-----------|
| **0** | ผ่านการตรวจสอบ |
| **1** | ข้อผิดพลาดทั่วไป |
| **2** | Guardrails violation - มีการแก้ไฟล์ที่ถูกป้องกัน |

---

## 💬 หมายเหตุ

- Hook นี้ทำงานเฉพาะกับ **staged files** เท่านั้น
- สามารถ **bypass** ได้ด้วย `--no-verify` (แต่ไม่แนะนำ)
- **AI จะเห็น error และต้องไปแก้ Code Logic จริงๆ**

## 📝 ตัวอย่างการใช้งาน

### ❌ ผิด (AI มักง่าย):

```javascript
// bad.test.js
test('should pass', () => {
  expect(true).toBe(true); // ✗ แก้ให้ผ่านซะ!
});
```

### ✅ ถูก (แก้ Code Logic):

```javascript
// sum.js
function sum(a, b) {
  return a + b; // แก้ bug จริงๆ
}

// sum.test.js
test('should add two numbers', () => {
  expect(sum(1, 2)).toBe(3); // Test ถูกต้อง
});
```
